# Reporte digital SCCALA

Aplicación web estática y responsiva para captura de reportes de mantenimiento, reparación e inspección de elevadores, escaleras eléctricas, rampas, montacargas y equipos similares.

## Uso
1. Abre `index.html` en un navegador moderno.
2. Llena el reporte desde celular, tableta o computadora.
3. Agrega fotografías y firmas.
4. Pulsa **Generar PDF** y selecciona **Guardar como PDF** en el diálogo de impresión.

## Publicar en GitHub Pages
1. Crea un repositorio nuevo.
2. Sube `index.html` y la carpeta `assets` a la raíz.
3. En Settings > Pages, selecciona Deploy from a branch y la rama `main`.

## Características
- Diseño optimizado para celular.
- Guardado automático local.
- Fotografías comprimidas en el navegador.
- Checklist editable con estados Correcto, Atención, Crítico y N/A.
- Actividades, hallazgos y recomendaciones dinámicas.
- Firmas digitales.
- Formato de impresión A4 con identidad SCCALA.

Nota: el borrador se guarda únicamente en el navegador/dispositivo utilizado. Para almacenamiento central, folios automáticos o envío directo por correo se requiere integrar una base de datos o servicio en la nube.
